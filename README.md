# SecurePasswordGenerator

A beginner-friendly Python program to generate **secure passwords** of customizable length and type.  
Supports alphanumeric, numeric, alphabetic, or alphanumeric with symbols.

## Features

- Generate passwords of **any length**.  
- Choose password type:  
  1. Alphanumeric (letters + numbers)  
  2. Numeric (numbers only)  
  3. Alphabetic (letters only)  
  4. Alphanumeric + Symbols (letters, numbers & special characters)  
- Handles invalid input gracefully with error messages.  
- Beginner-friendly and interactive.
